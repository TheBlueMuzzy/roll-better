  **Three changes needed to close the online game loop:**

  **A. Suppress local phase transition timers when online:**

  In App.tsx, the locking phase useEffect currently runs a 1-second timer that calls checkWinner/scoreRound or setPhase('unlocking'). For online games, the SERVER drives this transition via phase_change messages.

  Modify the locking useEffect:
  ```typescript
  useEffect(() => {
    if (phase === 'locking') {
      if (lastLockCount === 0) { playNoMatch(); }

      // Online: server drives phase transitions — don't use local timer
      if (isOnlineGame) return;

      // Offline: local 1s timer
      const timer = setTimeout(() => {
        if (checkWinner()) { scoreRound(); }
        else { setPhase('unlocking'); }
      }, 1000);
      return () => clearTimeout(timer);
    }
  }, [phase, setPhase, checkWinner, scoreRound, isOnlineGame, lastLockCount]);
  ```

  Also suppress the scoring and roundEnd useEffects when online (add `if (isOnlineGame) return;` at the top of each). These are Phase 18 concerns but we need to prevent them from firing on stale local state:
  - Scoring useEffect: add `if (isOnlineGame) return;` guard
  - RoundEnd useEffect: add `if (isOnlineGame) return;` guard

  **B. Wire unlock/skip to server in handleConfirmUnlock:**

  Modify handleConfirmUnlock in App.tsx. When `isOnlineGame`:
  - If player has selected dice: send `sendUnlockRequest(player.selectedForUnlock)` to server
  - If player skips (no selection, not must-unlock): send `sendSkipUnlock()` to server
  - Do NOT call local confirmUnlock/skipUnlock — server will broadcast unlock_result
  - Do NOT call startAIUnlockAnimations — server handles AI unlocks
  - After sending, clear selectedForUnlock locally for UI feedback: `useGameStore.getState().skipUnlock(0)` (just clears the selection array, doesn't change poolSize/lockedDice)

  const handleConfirmUnlock = useCallback(() => {

  **applyOnlineUnlockResult implementation:**
  ```typescript
  applyOnlineUnlockResult: (playerId, unlockedSlots, newPoolSize, serverLockedDice) => {
    const state = get();
    const playerIndex = state.onlinePlayerIds.indexOf(playerId);
    if (playerIndex === -1) return; // Bot or unknown

    const players = [...state.players];
    const player = { ...players[playerIndex] };
    player.lockedDice = serverLockedDice.map(ld => ({ goalSlotIndex: ld.goalSlotIndex, value: ld.value }));
    player.poolSize = newPoolSize;
    player.selectedForUnlock = [];
    players[playerIndex] = player;
    set({ players });
  }
  ```
