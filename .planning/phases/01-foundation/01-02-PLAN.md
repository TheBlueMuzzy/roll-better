<task type="auto">
  <name>Task 2: Add Rapier physics with test cube</name>
  <files>src/components/Scene.tsx</files>
  <action>
  1. Wrap the Scene contents with `<Physics>` from @react-three/rapier:
     - gravity={[0, -50, 0]} (faster gravity for snappy dice feel, from STATE.md decisions)
     - Note: default Rapier gravity is -9.81, we use -50 for game feel

  2. Wrap the floor plane mesh with `<RigidBody type="fixed">`:
     - Floor is static — things land on it but it doesn't move
     - Add a CuboidCollider or let auto-collider handle the plane

  3. Add a test cube above the floor:
     - `<RigidBody type="dynamic" position={[0, 5, 0]} restitution={0.3}>` (restitution from STATE.md)
       - Inside: `<mesh castShadow>`
         - BoxGeometry args={[1, 1, 1]}
         - MeshStandardMaterial color="#e74c3c" (red — visible placeholder)
     - The cube should fall under gravity, hit the floor, bounce slightly (restitution 0.3), and settle

  4. Verify physics integration:
     - Cube should NOT fall through the floor
     - Cube should bounce ~1-2 times then settle flat
     - No physics warnings in console

  Do NOT add dice-specific geometry — this is a plain cube to prove physics works.
  Do NOT add face detection — that's Phase 3.
  Do NOT add rolling impulse — just gravity drop to test collision.
  </action>
  <verify>
