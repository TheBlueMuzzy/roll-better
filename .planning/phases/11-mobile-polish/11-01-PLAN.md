**Constraining decisions:**
- Roll trigger: `handleRoll` in App.tsx calls `sceneRef.current?.rollAll()` when `phase === 'idle'`
- HUD shows "Tap To Roll" during idle phase — becomes "Shake to Roll" when shake enabled
- Settings toggle pattern: Zustand store field + toggle component in Settings panel
- CSS shake feedback (150ms, 90Hz) already used for rejected unlock — separate from device shake

**Discovery findings:**
- iOS 13+ requires `DeviceMotionEvent.requestPermission()` from user gesture → Promise('granted'|'denied')
- Android auto-grants — no permission needed
- Feature detect iOS: `typeof DeviceMotionEvent !== 'undefined' && typeof (DeviceMotionEvent as any).requestPermission === 'function'`
- Event: `devicemotion` → `event.accelerationIncludingGravity` ({x, y, z})
- Shake algorithm: magnitude of acceleration delta > threshold, with debounce

  <files>src/hooks/useShakeToRoll.ts</files>
  <action>
Create a custom React hook `useShakeToRoll(onShake: () => void, enabled: boolean)` that:

1. **Feature detection:** Check `window.DeviceMotionEvent` exists. Set `isSupported` accordingly. On desktop (no DeviceMotionEvent), return early with `isSupported: false`.

2. **iOS permission flow:**
   - Detect iOS: `typeof (DeviceMotionEvent as any).requestPermission === 'function'`
   - Expose `requestPermission()` async function that calls `DeviceMotionEvent.requestPermission()`
   - Track `permissionState`: 'prompt' | 'granted' | 'denied' | 'not-needed'
   - Android/non-iOS: set permissionState to 'not-needed' immediately
   - iOS: start as 'prompt', update after requestPermission resolves

3. **Shake detection algorithm:**
   - Listen to `devicemotion` event (only when enabled AND permission granted/not-needed)
   - Track last acceleration values (x, y, z) from `event.accelerationIncludingGravity`
   - Calculate delta magnitude: `sqrt(dx*dx + dy*dy + dz*dz)`
   - Threshold: 15 (tunable constant `SHAKE_THRESHOLD`)
   - Debounce: ignore triggers within 1000ms of last trigger (`SHAKE_COOLDOWN`)
   - On threshold exceeded + cooldown passed → call `onShake()`

4. **Cleanup:** Remove event listener on unmount or when disabled.

Return: `{ isSupported, permissionState, requestPermission }`

Do NOT use any external shake-detection libraries — the algorithm is simple enough to hand-roll. Use refs for acceleration tracking (not state — high-frequency events).
