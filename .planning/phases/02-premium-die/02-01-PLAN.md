<task type="auto">
  <name>Task 2: Add 3D pip dots to all 6 faces</name>
  <files>src/components/Die3D.tsx</files>
  <action>
Add pip dots as small CircleGeometry meshes positioned on each face of the die.

Standard Western d6 layout (opposite faces sum to 7):
- +Y (top): 1
- -Y (bottom): 6
- +X (right): 5
- -X (left): 2
- +Z (front): 3
- -Z (back): 4

Pip specs:
- CircleGeometry: radius 0.08, 16 segments
- Material: meshStandardMaterial, color "#ffffff" (white)
- Surface offset: position each pip 0.501 units from die center along the face normal (tiny offset avoids z-fighting)
- Rotation: each circle faces outward from the die face

Pip 2D positions (offset from face center, spacing 0.22):
- 1 pip: [[0, 0]]
- 2 pips: [[-0.22, 0.22], [0.22, -0.22]]
- 3 pips: [[-0.22, 0.22], [0, 0], [0.22, -0.22]]
- 4 pips: [[-0.22, 0.22], [0.22, 0.22], [-0.22, -0.22], [0.22, -0.22]]
- 5 pips: [[-0.22, 0.22], [0.22, 0.22], [0, 0], [-0.22, -0.22], [0.22, -0.22]]
- 6 pips: [[-0.22, 0.22], [0.22, 0.22], [-0.22, 0], [0.22, 0], [-0.22, -0.22], [0.22, -0.22]]

Create a helper to map 2D pip offsets to 3D positions based on face axis:
- +Y face: position [pipX, 0.501, pipY], rotation [-PI/2, 0, 0]
- -Y face: position [pipX, -0.501, -pipY], rotation [PI/2, 0, 0]
- +X face: position [0.501, pipY, -pipX], rotation [0, PI/2, 0]
- -X face: position [-0.501, pipY, pipX], rotation [0, -PI/2, 0]
- +Z face: position [pipX, pipY, 0.501], rotation [0, 0, 0]
- -Z face: position [-pipX, pipY, -0.501], rotation [0, PI, 0]

Performance: define CircleGeometry and pip MeshStandardMaterial ONCE outside the component (module-level constants), reuse for all ~21 pip meshes. Do NOT create new geometry/material per pip.

Do NOT use React state for pip positions — these are static constants defined at module level.
  </action>
