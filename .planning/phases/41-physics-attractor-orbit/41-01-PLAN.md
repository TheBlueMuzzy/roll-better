**Expose in useImperativeHandle:**
```typescript
setAttractTarget(target: [number, number, number] | null) {
  attractTargetRef.current = target;
  if (target && bodyRef.current) {
    bodyRef.current.wakeUp();
  }
}
```

**Add a `useFrame` hook** that applies spring-damped attractor forces when a target is set:

```typescript
useFrame((_, delta) => {
  const body = bodyRef.current;
  const target = attractTargetRef.current;
  if (!body || !target) return;

  // Clamp delta to prevent force spikes on tab-switch
  const dt = Math.min(delta, 0.05);

  const pos = body.translation();
  const vel = body.linvel();

  // Direction and distance to target
  const dx = target[0] - pos.x;
  const dy = target[1] - pos.y;
  const dz = target[2] - pos.z;
  const dist = Math.sqrt(dx * dx + dy * dy + dz * dz);

  if (dist < 0.01) return; // Close enough, skip

  // Spring force: F = stiffness * displacement
  const stiffness = 15;
  const damping = 8;
  const maxForce = 30;

  // Spring + damping (critically damped feel)
  let fx = stiffness * dx - damping * vel.x;
  let fy = stiffness * dy - damping * vel.y;
  let fz = stiffness * dz - damping * vel.z;

  // Clamp total force magnitude to prevent explosions
  const forceMag = Math.sqrt(fx * fx + fy * fy + fz * fz);
  if (forceMag > maxForce) {
    const scale = maxForce / forceMag;
    fx *= scale;
    fy *= scale;
    fz *= scale;
  }

  // Apply as impulse (force * dt for frame-rate independence)
  body.applyImpulse({ x: fx * dt, y: fy * dt, z: fz * dt }, true);

  // Suppress angular velocity during attraction for cleaner gathering
  const angvel = body.angvel();
  body.setAngvel(
    { x: angvel.x * 0.92, y: angvel.y * 0.92, z: angvel.z * 0.92 },
    true
  );

  // Keep body awake while attracting
  body.wakeUp();
});
```

**Important considerations:**
- The useFrame hook runs every frame even when no target is set — the early return (`if (!body || !target) return`) keeps it cheap.
- `stiffness=15, damping=8` gives a responsive-but-not-snappy feel. Phase 43 will tune these values.
- `maxForce=30` prevents extreme impulses when dice start far from target.
- Angular damping (`* 0.92`) slows spin during gathering for a controlled look, without zeroing it out completely.
- `wakeUp()` every frame ensures Rapier doesn't sleep the body while it's being attracted.
- Delta clamping (`Math.min(delta, 0.05)`) prevents huge forces when tab is backgrounded.
- Do NOT set `isRolling` during attraction — that's for the roll→settle cycle. Attraction is a separate concern.

**Add useFrame hook** inside the DicePool component (after the existing refs, before useImperativeHandle):

```typescript
// Read gather state reactively at top of component (outside useFrame)
const gatherActive = useGameStore((s) => s.gatherState.active);
const gatherTouchPosition = useGameStore((s) => s.gatherState.touchPosition);
const gatherDieCount = useGameStore((s) => s.gatherState.dieCount);

useFrame((_, delta) => {
  // Detect gathering start/stop transitions
  if (gatherActive && !wasGatheringRef.current) {
    // Just started gathering — reset elapsed time
    gatherElapsedRef.current = 0;
    rotationOffsetRef.current = 0;
    wasGatheringRef.current = true;
  } else if (!gatherActive && wasGatheringRef.current) {
    // Just stopped gathering — clear all attract targets
    wasGatheringRef.current = false;
    for (let i = 0; i < count; i++) {
      dieRefs.current[i]?.setAttractTarget(null);
    }
    return;
  }

  if (!gatherActive || !gatherTouchPosition) return;

  // Clamp delta
  const dt = Math.min(delta, 0.05);
  gatherElapsedRef.current += dt;

  // Rotation speed ramps from 0.5 to 4.0 rad/s over 3 seconds
  const rampT = Math.min(gatherElapsedRef.current / 3.0, 1.0);
  const rotationSpeed = 0.5 + rampT * 3.5; // 0.5 → 4.0
  rotationOffsetRef.current += rotationSpeed * dt;

  // Compute orbital goal positions
  const goals = getGatherPoints(
    gatherTouchPosition,
    count,           // use actual pool count, not gatherDieCount (they should match but pool count is authoritative)
    undefined,       // default radius
    rotationOffsetRef.current
  );

  // Assign each die its goal point
  for (let i = 0; i < count && i < goals.length; i++) {
    dieRefs.current[i]?.setAttractTarget(goals[i]);
  }
});
```

**Handle edge cases:**
- If `count` changes during gathering (shouldn't happen, but defensive): the loop is bounded by `Math.min(count, goals.length)`.
- When gathering stops, explicitly null out all attract targets so dice fall back to the floor naturally via gravity.
- The `wasGatheringRef` prevents re-running the stop logic every frame when not gathering.

**Important — don't reset physics state:** When gathering stops, do NOT reset die positions or call roll(). Just remove the attractor. Dice will fall back under gravity. Phase 42 will add momentum-based release.

**Reset on rollAll:** In the existing `rollAll()` imperative method, also clear gathering state:
```typescript
// Inside rollAll():
wasGatheringRef.current = false;
gatherElapsedRef.current = 0;
rotationOffsetRef.current = 0;
```
This ensures a roll doesn't fight with stale attractor targets.
