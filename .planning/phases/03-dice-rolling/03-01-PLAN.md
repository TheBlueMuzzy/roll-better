Create `src/components/PhysicsDie.tsx`:

1. Component wraps Die3D inside a RigidBody with CuboidCollider (args [0.5, 0.5, 0.5]).
2. Accept props: `color` (string), `position` (initial spawn position), `onSettle` callback (fires when die stops), `onResult` callback (placeholder for Phase 3 Plan 2).
3. Use `useImperativeHandle` with `forwardRef` to expose a `roll()` method on the ref. The ref type should be a custom `PhysicsDieHandle` interface exported from this file.
4. `roll()` method:
   - Reset die position to its spawn position using `setTranslation()`, reset velocity with `setLinvel({x:0,y:0,z:0})` and `setAngvel({x:0,y:0,z:0})`
   - Set random initial rotation using `setRotation()` with a random quaternion (use Three.js Quaternion.random() or Euler with random angles 0-2PI on all axes)
   - Wake the body with `wakeUp()`
   - Apply upward impulse with random horizontal offset: `applyImpulse({x: randRange(-3,3), y: randRange(15,25), z: randRange(-3,3)}, true)` — the horizontal offset creates natural tumbling
   - Apply random torque impulse: `applyTorqueImpulse({x: randRange(-5,5), y: randRange(-5,5), z: randRange(-5,5)}, true)` — adds spin variety
5. RigidBody props: `type="dynamic"`, `ccd`, `restitution={0.35}`, `friction={0.5}`, `angularDamping={0.3}`, `linearDamping={0.1}`
6. Use `RapierRigidBody` type from `@react-three/rapier` for the rigid body ref.
7. Helper: `randRange(min, max)` returns random float in range.

Update `src/components/Scene.tsx`:

- Add a click handler on the floor mesh: `onClick` calls `physicsDieRef.current.roll()`.

- Set PhysicsDie initial position to `[0, 1, 0]` (resting near floor).

<task type="auto">
  <name>Task 2: Add settle detection via sleep events</name>
  <files>src/components/PhysicsDie.tsx</files>
  <action>
Add settle detection to PhysicsDie:

1. Track rolling state with a ref (NOT React state — avoid re-renders during physics): `const isRolling = useRef(false)`.
2. In `roll()`, set `isRolling.current = true`.
3. Add `onSleep` prop to RigidBody: when fired AND `isRolling.current` is true, set `isRolling.current = false` and call `onSettle?.()` callback.
4. Add `onWake` prop to RigidBody: if the die wakes back up after sleeping (e.g., bumped by another die), set `isRolling.current = true` again. This prevents false settle signals.
5. Export the `PhysicsDieHandle` interface with: `roll(): void` and `isSettled: boolean` (getter that returns `!isRolling.current`).

Do NOT use React useState for isRolling — it would cause unnecessary re-renders during physics simulation. Use refs only.

  <done>onSettle callback fires reliably when die stops after a roll. Re-rolling resets the state. No false settles during active rolling.</done>
