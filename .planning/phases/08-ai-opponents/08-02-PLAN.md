**Multi-player turn flow (designed):**
1. initRound: generate shared goal, reset all players, set currentPlayerIndex=0
2. Current player's turn: idle → rolling → locking → unlocking → idle (loop)
3. When current player locks all 8 → scoring for that player
4. After scoring → advance to next player (reset their dice, same goal) OR roundEnd if all played
5. After all players scored → roundEnd → handicap → new round with new goal
6. Each player plays the SAME goal, scored independently on efficiency
