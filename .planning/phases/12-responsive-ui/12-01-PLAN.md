<task type="auto">
  <name>Task 1: Create responsive CSS custom properties</name>
  <files>src/App.css</files>
  <action>
Add a `:root` block at the top of App.css (after any existing reset/body styles) defining CSS custom properties using clamp() formulas. These become the responsive vocabulary for all UI:

**Font sizes:**
- `--fs-xs`: clamp(10px, 2.5vw, 13px) — stats, hints
- `--fs-sm`: clamp(12px, 3vw, 14px) — labels, secondary text
- `--fs-md`: clamp(14px, 3.5vw, 18px) — body text, buttons
- `--fs-lg`: clamp(16px, 4vw, 22px) — headings, primary actions
- `--fs-xl`: clamp(20px, 5vw, 28px) — titles

**Spacing:**
- `--sp-xs`: clamp(4px, 1vw, 8px)
- `--sp-sm`: clamp(8px, 2vw, 14px)
- `--sp-md`: clamp(12px, 3vw, 20px)
- `--sp-lg`: clamp(16px, 4vw, 28px)
- `--sp-xl`: clamp(24px, 5vw, 36px)

**Layout:**
- `--overlay-max-w`: clamp(280px, 85vw, 420px) — modal/overlay max width
- `--btn-max-w`: clamp(240px, 75vw, 360px) — primary action button max width
- `--touch-target`: clamp(40px, 10vw, 56px) — minimum touch target
- `--border-radius`: clamp(8px, 2vw, 16px)

Do NOT change any existing styles in this task — only add the custom properties block. This keeps the commit atomic and testable.
  </action>
  <verify>Open src/App.css and confirm the :root block exists with all custom properties defined. Run `npm run build` to ensure no syntax errors.</verify>
  <done>:root block with 15+ CSS custom properties using clamp() formulas. Build passes.</done>
