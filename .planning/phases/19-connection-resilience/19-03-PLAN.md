<objective>
Add visual feedback for connection state changes during online games — a "Reconnecting..." overlay when disconnected and a toast notification when another player reconnects.

Purpose: Without this, a player whose connection drops sees no feedback — the game just freezes silently. Other players don't know a teammate dropped and came back. This plan provides clear visual communication.
Output: Reconnecting overlay shown during disconnect, toast for player_reconnected events.
</objective>

**Key architecture facts:**
- `useRoom` is called inside `LobbyScreen`, which is always rendered (visibility toggle, not conditional mount). So `useRoom` stays alive during the game.
- `useRoom` sets `isConnected = false` on game-active disconnect and `true` on reconnect (19-02).
- `isConnected` is local React state inside `useRoom` — NOT accessible from App.tsx or other components.
- `useOnlineGame` handles `player_reconnected` message with `msg.playerName` and `msg.playerId` — currently just logs.
- The game container renders Canvas, HUD, and TipBanner. Overlays are HTML divs alongside the Canvas.

**Approach — surface connection state via Zustand:**
- Add `isOnlineDisconnected: boolean` to gameStore (default false).
- `useRoom` sets it true on game-active disconnect, false on reconnect/rejoin.
- App.tsx reads it to render overlay.
- For player_reconnected toast: useOnlineGame dispatches CustomEvent, App.tsx listens with local state.
- No new component files — inline in App.tsx (small enough).

**App.tsx — Render overlay + toast:**

Add store selector and local state:
```typescript
const isOnlineDisconnected = useGameStore((s) => s.isOnlineDisconnected);
const [reconnectToast, setReconnectToast] = useState<string | null>(null);
```

Add effect for reconnect toast events:
```typescript
useEffect(() => {
  const handler = (e: Event) => {
    const name = (e as CustomEvent).detail.name;
    setReconnectToast(`${name} reconnected`);
    setTimeout(() => setReconnectToast(null), 3000);
  };
  window.addEventListener('player-reconnected', handler);
  return () => window.removeEventListener('player-reconnected', handler);
}, []);
```
