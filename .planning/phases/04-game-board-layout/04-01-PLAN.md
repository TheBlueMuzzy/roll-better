  Create a GoalRow component that renders 8 static Die3D instances in a horizontal line.

  - Slot spacing: 0.7 (center-to-center) — gives 0.041 gap between die edges
  - X positions: centered at 0, from -2.45 to +2.45 in steps of 0.7

  - Y position: DIE_SIZE / 2 (die rests on floor surface)

  - `SLOT_COUNT = 8`
  - `getSlotX(index: number): number` — returns X position for slot index 0-7

  Each die is a `<group>` with `<Die3D />` scaled to DIE_SIZE, positioned at the slot X. Use white/cream color (Die3D default '#e8e0d4') for Goal dice — do NOT use player colors.

  These are purely visual (no physics). Render as a `<group>` — NOT inside a RigidBody or Physics component.

  Do NOT add rotation or animation — dice sit flat with face showing the value. To show a specific face value pointing up, apply a pre-computed rotation quaternion that puts the desired face on top.
