## Decisions Made

- depthTest: false to prevent z-fighting with floor plane
- Y-offset layering: circle at 0.05, star at 0.06, text at 0.07

## Deviations from Plan

### Auto-fixed Issues

**1. Floor z-fighting — elements invisible at y=0**
- **Found during:** Task 1 visual verification
- **Issue:** Circle and text at y=0 were hidden by floor mesh at y=0.001
- **Fix:** Raised all elements (y=0.05+) and set depthTest: false
- **Committed in:** 2ac74b5

**2. Star unicode glyph not rendering in drei/Text**
- **Found during:** Task 1 visual verification
- **Issue:** Default Inter font doesn't include ★ character
- **Fix:** Used THREE.ShapeGeometry with a 5-point star path instead
- **Committed in:** 2ac74b5

**3. Overlapping first goal die**
- **Found during:** Checkpoint review by Muzzy
- **Issue:** 3D elements centered on anchor point instead of right-edge aligned (HTML used translate(-100%))
- **Fix:** Added X_SHIFT constant, then increased to -1.125 per Muzzy feedback
- **Committed in:** c0ef77f, edf68e4
