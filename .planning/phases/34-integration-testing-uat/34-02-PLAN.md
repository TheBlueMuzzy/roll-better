    **Setup:** Start a game with 2+ human players (or 1 human + bots so there ARE bot seats available).

    ### Scenario A: Late friend joins mid-game
    1. Game is in progress (any phase — rolling, locking, unlocking)
    2. Open a NEW browser window/tab, go to the game URL
    3. Enter the active room code and join
    4. Confirm: MainMenu shows mid-game join UI (not lobby)
    5. Confirm: Available bot seats are displayed for selection
    6. Tap a bot seat to claim it
    7. Confirm: Waiting UI appears with spinner + "Seat claimed! Joining next round..." or similar
    8. Wait for phase boundary
    9. Confirm: Player transitions into the game, sees correct game state
    10. Confirm: Other players see "X joined the game" notification (not "X is back!")
    11. Confirm: New player can roll, lock, unlock normally

    ### Scenario B: Room full — no bot seats available
    1. Start a game where ALL seats are occupied by humans (no bots)
    2. Open a new browser window, try to join the room code
    3. Confirm: Error message appears with "Room is full" or similar
    4. Confirm: TRY AGAIN button is visible and functional
    5. Tap TRY AGAIN — confirm it re-attempts the join (useful if a player leaves in the meantime)
