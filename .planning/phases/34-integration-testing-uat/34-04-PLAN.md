    **Setup:** Create a room with 2-3 human players (use multiple browser windows or PC + phone).

    ### Scenario A: Host leaves — migration to next human
    1. Identify which player is the host (created the room)
    2. Mid-game, close the HOST's browser tab entirely
    3. Confirm: Game continues for remaining players (no freeze, no error)
    4. Confirm: One of the remaining human players becomes the new host
    5. Confirm: Host's old seat transitions to bot behavior
    6. Play a few more rounds to confirm game is stable under new host

    ### Scenario B: Rage quit — non-host permanent leave
    1. With 2+ human players in a game, close a NON-HOST player's tab
    2. Confirm: That player's seat transitions to bot
    3. Confirm: Game continues normally for remaining players
    4. Confirm: Host remains unchanged
    5. Confirm: Notification shows on other players' screens (seat state change)

    ### Scenario C: All humans leave — room dissolution
    1. Start a game with 2 human players + bots
    2. Close Player 2's tab (becomes bot)
    3. Close Player 1's tab (last human leaves)
    4. Confirm: Room dissolves (room_closed broadcast sent)
    5. If any mid-game joiners were waiting, confirm they receive room_closed
    6. Try to rejoin the room code — confirm it's no longer active

    Report which scenarios passed/failed and any unexpected behavior.
